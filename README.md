```
Criando a tabela usuarios

Agora há uma restrição! Somente usuários logados podem gerenciar o cadastro de alunos, então você precisará criar uma tabela “usuarios” e criar um CRUD
```

### Atividade 2 - Criando um CRUD

```
Você criará um CRUD para nosso sistema de cadastro de alunos. Portanto, você terá que listar todos alunos 
e permitir a edição deles, inclusão de novos, exclusão e simplesmente mostrar os dados de um aluno.
```

### Atividade 1 - Criando um arquivo .SQL
```
Criando um arquivo .sql

Crie um arquivo .sql para colocar a criação de seu banco de dados e tabelas para facilitar futuramente a 
correção do seu projeto, então o arquivo ficará assim:

- CREATE DATABASE ...

- CREATE TABLE ...

- INSERT INTO ...

    Você criará um banco de dados no MYSQL e uma tabela “alunos” com id (chave primária), 
    nome (varchar(255)), nota (int) e inserirá 20 registros variados na    
    tabela “alunos”, coloque estes 20 inserts no nosso arquivo .sql. Crie um sistema    
    para listar todos os alunos e também para listar os que têm as três maiores notas.
```
